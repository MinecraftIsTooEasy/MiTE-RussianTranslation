# MiTE-RussianTranslation
Russian translation of MiTE and its addons.
Перевод MiTE на русский язык, сделанный с душой.
Убедительная просьба не выдавать перевод за свой и не продавать его. Вы можете использовать перевод где угодно, только не забудьте оставить ссылку на оригинал и/или указать автора.

## For ModDevs
You can request translation for your mod or just take it from this pack, but you must leave an author nickname or link to original pack.
